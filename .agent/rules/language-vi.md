# 🇻🇳 Ngôn ngữ giao tiếp - Tiếng Việt

## Quy tắc ngôn ngữ

- **Luôn trả lời và giải thích bằng tiếng Việt** trong mọi cuộc hội thoại với người dùng.
- Sử dụng tiếng Việt cho: phân tích, hướng dẫn, tóm tắt, câu hỏi, và mọi nội dung giao tiếp.
- **Code vẫn viết bằng tiếng Anh**: tên biến, tên hàm, class, comment trong code, commit message, và tên file đều giữ nguyên tiếng Anh theo chuẩn lập trình quốc tế.
- Artifact (plan, walkthrough, report) viết bằng tiếng Việt khi nội dung hướng đến người dùng.
