ADB Wireless Connection Guide

Follow these steps on your phone to connect to this machine wirelessly.

1
Connect to same WiFi
Make sure your phone and this computer are connected to the exact same Wi-Fi network.
2
Enable Developer Options
Go to Settings > About Phone. Tap Build Number 7 times until you see "Developer mode enabled".
3
Turn on Wireless Debugging
Go to Settings > System > Developer Options. Enable Wireless Debugging.
4
Find Pairing Info
Tap on "Wireless Debugging" to enter settings. Choose "Pair device with pairing code". Note down the IP address, Port, and 6-digit Code.
5
Submit on PC Web App
Enter the IP, Port, and Code in the PC browser dashboard to complete the wireless pairing process.
